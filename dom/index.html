O que é DOM?

DOM é a sigla para Document Object Model e é um modelo utilizado pelo navegador web para interpretar uma página. Para isso, utiliza-se o objeto document, o qual permite que o navegador acesse a página. É possível, utilizando document, acessar propriedades, tais como documentElement, getElementsById e appendChild. O modelo DOM interpreta a Página tal qual uma árvore, utilizando-se dessa estrutura para organizar as marcações HTML.
